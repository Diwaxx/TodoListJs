# Todo List

Список задач на `React + JavaScript + Vite`, который загружает задачи из `JSONPlaceholder`, показывает первые 15 записей на главном экране, поддерживает поиск и фильтрацию по статусу, а также открывает отдельную страницу с деталями задачи.

## Как запустить

```bash
npm install
npm run dev
```

Для проверки сборки:

```bash
npm run build
```

Для проверки линтера:

```bash
npm run lint
```

## Что использовано

- `React`
- `JavaScript`
- `Vite`
- `react-router-dom`
- `CSS`

- вынести сообщения об ошибках и загрузке в более универсальные UI-блоки
